算法训练——存在重复元素 难度——★✰✰✰✰

算法练习题

Nov 17, 2024
阅读时长: 1 分钟

题目要求

给你一个整数数组 nums 。如果任一值在数组中出现 至少两次 ，返回 true ；如果数组中每个元素互不相同，返回 false 。

示例 1：

输入：

nums = [1,2,3,1]

输出：

true

解释：

元素 1 在下标 0 和 3 出现。

示例 2：

输入：

nums = [1,2,3,4]

输出：

false

解释：

所有元素都不同。

示例 3：

输入：

nums = [1,1,1,3,3,4,3,2,4,2]

输出：

true

提示：

  • 1 <= nums.length <= 105
  • -109 <= nums[i] <= 109

解题过程

思路

只要将数组进行排序，然后依次比较相邻的元素就可以了。

（ps：使用二次循环也是可以的，不过一般情况下会超时）

int cmp(const void* _a, const void* _b) {
    int a = *(int*)_a, b = *(int*)_b;
    return a - b;
}

bool containsDuplicate(int* nums, int numsSize) {
    qsort(nums, numsSize, sizeof(int), cmp);
    for (int i = 0; i < numsSize - 1; i++) {
        if (nums[i] == nums[i + 1]) {
            return true;
        }
    }
    return false;
}

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        for (int i = 0; i < n - 1; i++) {
            if (nums[i] == nums[i + 1]) {
                return true;
            }
        }
        return false;
    }
}
